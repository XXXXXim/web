CSS 속성이 워낙 많아서 처음엔 뭐가 뭔지 헷갈리기 쉽지! 하지만 진짜 다행인 건, **실무에서 매일 쓰는 속성은 전체의 20% 정도**밖에 안 된다는 거야.

그 핵심 20%를 카테고리별로 꽉꽉 눌러 담아서 정리해 줄게. 이것만 알면 웹페이지 디자인의 90%는 네 마음대로 주무를 수 있어! 😎

---

### 📝 1. 글자(Text) 꾸미기

글자의 크기, 색상, 정렬을 담당하는 가장 기본적인 속성들이야.

| 속성 (Property) | 무슨 역할? | 실전 예시 |
| --- | --- | --- |
| **`color`** | 글자 색상 변경 | `color: blue;` 또는 `color: #ff0000;` |
| **`font-size`** | 글자 크기 | `font-size: 16px;` 또는 `font-size: 2rem;` |
| **`font-weight`** | 글자 굵기 | `font-weight: bold;` (또는 100~900 숫자) |
| **`text-align`** | 텍스트 가로 정렬 | `text-align: center;` (가운데), `left`, `right` |
| **`line-height`** | 줄 간격 (위아래 여백) | `line-height: 1.5;` (글자 크기의 1.5배) |
| **`text-decoration`** | 밑줄, 취소선 긋기/없애기 | `text-decoration: none;` (a 태그 밑줄 없앨 때 필수!) |

---

### 📦 2. 박스 크기와 여백 (Box Model)

HTML의 모든 요소는 '네모난 상자(Box)'야. 이 상자의 덩치와 위치를 정하는 아주 중요한 녀석들이지.

| 속성 (Property) | 무슨 역할? | 실전 예시 |
| --- | --- | --- |
| **`width`** | 상자의 가로 너비 | `width: 300px;` 또는 `width: 100%;` |
| **`height`** | 상자의 세로 높이 | `height: 200px;` |
| **`margin`** | **바깥 여백** (남을 밀어냄) | `margin: 20px;` (상하좌우 모두 20px 띄움) |
| **`padding`** | **안쪽 여백** (내 뱃살을 찌움) | `padding: 10px;` (테두리와 내용물 사이 공간) |
| **`box-sizing`** | 테두리와 패딩을 너비에 포함시킬지 결정 (⭐필수) | `box-sizing: border-box;` (이거 안 쓰면 박스 크기 계산할 때 머리 터짐) |

---

### 🎨 3. 테두리와 배경

박스를 예쁘게 색칠하고 테두리를 깎는 속성들이야.

| 속성 (Property) | 무슨 역할? | 실전 예시 |
| --- | --- | --- |
| **`background-color`** | 배경 색상 | `background-color: lightgray;` |
| **`border`** | 테두리 (두께, 선종류, 색상) | `border: 1px solid black;` (1px 실선 검정색) |
| **`border-radius`** | 테두리 모서리 둥글게 깎기 | `border-radius: 10px;` (버튼 예쁘게 만들 때 필수!) |
| **`box-shadow`** | 박스에 그림자 주기 | `box-shadow: 2px 2px 5px gray;` |

---

### 🏗️ 4. 배치와 레이아웃 (끝판왕)

요소들을 가로로 나열하거나, 원하는 위치에 공중부양 시키는 마법의 속성들이야.

| 속성 (Property) | 무슨 역할? | 실전 예시 |
| --- | --- | --- |
| **`display`** | 요소의 성질을 바꿈 (블록 ↔ 인라인) | `display: flex;` (요즘 가로 정렬은 무조건 이거 씀!) |
| **`position`** | 요소를 원하는 위치에 콕 박아둠 | `position: absolute;` (절대 위치), `fixed` (스크롤 해도 화면에 고정) |
| **`top`, `left`, `right`, `bottom**` | `position`을 썼을 때, 상하좌우에서 얼마나 떨어질지 정함 | `top: 50px; left: 20px;` |
| **`z-index`** | 요소들이 겹쳤을 때 누가 위로 올라올지 순서 | `z-index: 99;` (숫자가 클수록 화면 맨 앞으로 옴) |
